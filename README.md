# AWS VPC creation via Cloudformation - Scenario 1

This Repo contains the diagram, Cloudformation code and relevant information to build an AWS VPC along with these elements:

- Security Groups
- NACL
- Subnets (Public and Private)
- IGW
- Route Table
- EC2 Instances
